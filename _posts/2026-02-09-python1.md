> phython의 자료형 문법

## 수 number

`정수형 integer number` - 정수를 표현합니다\
`실수형 float number` - 기존 정수에서 소수점을 추가로 표현합니다

```phython
a = 12      # 정수형
b = 2.718   # 실수형

c = a + b   # 정수형과 실수형을 더할 수 있음
```

## 문자열 string

```phython
# stringname = 'text'

a = 'text1'
b = "text2"
c = '''text3'''
d = """text4"""
```

> 문장내에 `를 사용하고 싶다면, 문자열을 ",''',"""로 감쌉니다
{: .prompt-info }

```phython
a = " '를 사용"
```

## 리스트 list

`리스트 list` - 순서를 정해서 관리하는 자료형

```phython
# listname = [ data1,data2,data3,data4 ]  

l1 = [1,1,2,4]
l2 = ['a','b','c']
l3 = [1,3,'a','c']
l4 = [1,2,[1,2,3]]
```
\
`list1 + list2` - 리스트의 병합

```phython
list1 = [ 0,1,'a',4 ]  
list2 = [ 5,'b','d' ]

print( list1 + list2 )      # [ 0,1,'a',4,5,'b','d' ]
print( list2 + list1 )      # [ 5,'b','d',0,1,'a',4 ]  
```

\
`list * integernumber` - 리스트의 반복

```phython
list = [ 0,1,'a',4 ]  

print( list * 3 )      # [ 0,1,'a',4,0,1,'a',4,0,1,'a',4 ]
```

\
`listname[integernumber]` - 리스트의 number번째 index에 위치한 data

```phython
listname = [1,2,3,4]

a = listname[0]     # a의 값은 1
b = listname[1]     # b의 값은 2
c = listname[2]     # c의 값은 3
d = listname[3]     # d의 값은 4
```

>음수번째 index는 역행을 의미합니다

```phython
listname = [1,2,3,4]

a = listname[-4]     # a의 값은 1
b = listname[-3]     # b의 값은 2
c = listname[-2]     # c의 값은 3
d = listname[-1]     # d의 값은 4
```

>리스트의 내용을 수정할수 있습니다

```phython
listname = [1,2,3,4]

listname[2] = 10    # 리스트의 3번째값은 10
print(listname)     # listname = [1,2,10,4]
```
