## 시제 Tense

> 사건의 시간적 시점을 표시하는 문법 범주

`발화시점 Speech Time` - 화자가 문장을 말하는 시점 \
`기준시점 Reference Time` - 화자가 사건을 이야기하기 위해 설정한 시간적 중심점 \
`사건시점 Event Time` - 실제 사건이 일어난 시간

> 영어의 시제는 2시제 혹은 3시제로 설명됩니다
{: .prompt-info }

3시제설\
`현재 Present` - 사건시점이 발화시점과 동일 (E=S) \
`과거 Past` - 사건시점이 발화시점보다 과거 (E→S)\
`미래 Future` - 사건시점이 발화 시점보다 미래 (S→E)

2시제설\
`현재 Present` - 사건시점이 발화시점과 동일 \
`과거 Past` - 사건시점이 발화시점보다 과거 

> 2시제가 미래를 포함하지 않는 것은 미래를 나타낼 때 동사의 굴절이 없기 때문입니다
{: .prompt-info }
