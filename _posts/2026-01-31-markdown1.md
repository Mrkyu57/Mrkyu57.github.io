> 깃허브 블로그를 작성할 때 필요한 Github-Flavored Markdown의 문법

## 머릿말

```markdown
---
title: "글의 제목"
date: XXXX-XX-XX XX:XX:XX +0900
categories: [상위 카테고리, 하위 카테고리]
tags: [태그1, 태그2]
---
```

## 제목

```markdown
# 1단계 제목
## 2단계 제목
### 3단계 제목
#### 4단계 제목
##### 5단계 제목
###### 6단계 제목
```

> #과 제목 사이에 반드시 공백이 존재해야 합니다
{: .prompt-tip }

## 인용문

```markdown
> 인용문
```

> >과 인용문 사이에 반드시 공백이 존재해야 합니다
{: .prompt-tip }

## 프롬프트

```markdown
> 빨간색 프롬프트 내에 사용할 문장
{: .prompt-danger }

> 노란색 프롬프트 내에 사용할 문장
{: .prompt-warning }

> 초록색 프롬프트 내에 사용할 문장
{: .prompt-tip }

> 파란색 프롬프트 내에 사용할 문장
{: .prompt-info }
```

##  **굵은 글자**

```markdown
기본 글자 **굵은 글자**
```


## *기울임 글자*

```markdown
기본글자 *기울임 글자*
```
